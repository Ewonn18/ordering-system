# 🛒 Ordering System (Laravel + Livewire)

A web-based ordering system designed to streamline product browsing, order placement, and order management. This project demonstrates full-stack development using Laravel and Livewire with role-based access control for Admin and Customers.

---

## 🚀 Features

### 👤 Customer Side

- Browse available products
- Add items to cart
- Place orders
- View order status in real-time

### 🛠️ Admin Side

- Manage products (CRUD)
- View and update customer orders
- Order status management

---

## 🧑‍💻 Tech Stack

- Laravel (PHP)
- Livewire
- MySQL
- Tailwind CSS

---

## ⚙️ Installation

```bash
git clone https://github.com/Ewonn18/ordering-system.git
cd ordering-system
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
